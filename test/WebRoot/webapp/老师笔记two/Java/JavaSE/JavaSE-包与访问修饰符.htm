包(package)

  • 用于将完成不同功能的类分门别类，放在不同的目录(包)下。
  • 包的命名规则：每个字母都用小写，将公司域名反转作为包名 www.xxx.com(com.xxx 包名)。
  • 如果在定义类的时候没有使用package，那么java就认为我们所定义的类的位置在默认包里面(default package)。
  • 自动生成包，javac -d . PackageTest.java。
  • aa.bb.cc和aa.bb.cc.dd我们称后者是前者的子包。

导入(import)：将package分离的类导入回来

  • 用于将完成不同功能的类分门别类，放在不同的目录(包)下。
  • import aa.*; import aa.bb.*; import aa.bb.Test.java。
  • 如果引用的类和当前类在同一包中，那么就没有必要import
  • 在当前类中引用其他包中的类，必须要使用import

关于package、import、class的顺序问题

  • 首先需要定义包(package)，可选
  • 然后使用(import)进行导入，可选
  • 最后是class、interface定义类

访问修饰符：

  • public(公共的)：
    • 被public修饰的属性和方法可以被所有类访问。
  • protected(受保护的)：
    • 被protected修饰的属性和方法可以在类内部、相同包、以及该类的子类内部(直接使用)所访问。
  • private(私有的)：
    • 被private修饰的属性和方法只能在该类内部使用。
  • default(没有加任何访问修饰符，默认)：
    • 修饰的属性和方法在类类部、以及相同包下面的类可以使用。
