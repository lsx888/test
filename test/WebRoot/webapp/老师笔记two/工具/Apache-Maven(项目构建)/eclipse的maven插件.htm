eclipse的maven插件

最后修改时间：2016-02-22 17:49:58

准备工作

目前最新版本的eclipse已经自带maven插件，如果使用的旧版本的eclipse可以自己下载maven插件，下载和安装这里不做讲解。

请确保已经在本地安装了maven。关于maven的安装，请看第一节的内容。

插件配置

  1. 检查eclipse的maven插件是否安装成功，如图：
    若没有安装maven插件，我们需要先安装maven插件。
  2. 配置maven安装目录，依次打开Window –> Perferences –> Maven ，展开Maven的配置界面，如下图：
    然后点击Installations –> add 选择maven安装目录，这里我的Maven安装目录为D:\maven\apache-maven-3.2.3，选择你的Maven安装目录，并点击确定, 之后可以点击Apply,点击OK，即可完成
  3. 然后， 我们在Maven的配置界面，设置User Settings Global Settings选择maven 安装目录下conf文件夹下的settings.xml，这里我的Maven安装目录为D:\maven\apache-maven-3.2.3\conf\settings.xml，选择你的Maven安装目录，检查Local Repository 项，如果为D:/maven/repository则配置成功，否则重新配置上一步。
    到此eclipse就配置好了

使用插件创建项目

  1. 我们在Eclipse菜单栏中点击File->New->Other->Maven,在弹出的对话框中会看到，如下图所示：
  2. 选择Maven Project，请选中Create a simple project(skip archetype selection),之后点击Next
  3. 填写Group id和Artifact id， Version默认，Packaging默认为jar,Name，Description选填，其他的不填
  4. 之后点击Finish即可
    前三步就可以创建一个简单的maven项目，如果我们想创建一个Maven的web项目，把第三步的Packaging的类型改为war，之后点击Finish即可

使用插件执行mvn命令

在pom文件上右键，run as 会出现某些mvn命令，但是如果我们要执行一些其他命令呢，使用run as ->maven build...如下图：
在Goals中填写对应的执行阶段即可，如：clean test或clean install，填好后点击run执行即可
