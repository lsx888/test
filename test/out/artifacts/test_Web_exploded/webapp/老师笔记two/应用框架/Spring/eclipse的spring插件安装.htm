eclipse的spring插件安装

最后修改时间：2016-02-22 16:18:48

简介

Spring Tool Suite简称STS，spring的eclipse插件，其具有许多方便使用spring开发的许多功能。注意：该插件不是使用spring框架的必须应用。其作用就像eclipse之于java。eclipse的插件安装有许多方式，如：1.直接下载相关插件包，直接解压安装。2. Help->Install New Software，使用对应uri安装或下载包安装。3.Help->eclipse Marketplace，输入关键字搜索安装。这里我们讲第二种方式的两种安装方法。

使用zip包安装

1.首先进入STS页面，http://spring.io/tools/sts/all，如下图：
请根据eclipse的版本选择对应的zip包进行操作，这里我选择4.5版本（4.5对应你的eclipse的版本）
2.点击 Help->Install New Software进入页面后再点击add进入如下页面
3.点击Archive，选择对应下载的zip包
4.点击确定后出现如下内容，如图选择所有有IDE的项目
