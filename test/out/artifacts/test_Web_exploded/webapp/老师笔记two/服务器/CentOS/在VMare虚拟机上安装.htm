在VMare虚拟机上安装

最后修改时间：2015-08-20 11:39:33

一、下载安装镜像文件

这里安装CentOS-7-x86_64-Minimal-1503-01.iso(注意这是一个64位的，根据实际情况选择你的安装镜像)。这个自带安装软件最少，简单而小巧，偏于你了解安装和部署开发环境，所以选择这个版本，作为linux开发者，熟悉命令行界面是最基本的。关于镜像文件的下载和版本这里不做讲解。百度一下啥都有。

二、VMare新建虚拟机

VMware 新建一个linux虚拟机，根据需要配置好各项设置，都是傻瓜安装，不多讲。以下是新建虚拟机的过程：

1.新建虚拟机
2.选择典型

注意：下面这一步（图中）选择最后一项。在虚拟机新建后，再配置安装镜像的路径。（否则，你安装完后，系统起不来，很多人都遇到过这冲情况。）

3.选择跳过

　如果你选择第二项，硬件设置中多了一个CD/DVD(IDE)，装载了autoinst.iso，另一个CD/DVD 2(IDE)装载了安装镜像CentOS-6.3-x86_64-minimal.iso，重新建了个虚拟机，发现在选择安装镜像文件那里，有个提示“CentOS 64-bit detected.This operating system will use Easy Install 。就是这里导致的。所以这里强调：在安装镜像选择那里，选择“I will install the operating system later.“，创建完后，硬件设置中设置安装镜像。

4.选择安装的系统

注意这里选择的系统名称和版本，一定要和镜像文件对应。

5.选择安装路径

注意这里最好不要中文路径。（没验证过，职业病）。虚拟机名字。随便填写，如果你安装了多个linux，那就最好填写不同的名字以便区分。

6.指定磁盘空间

这里了可以直接默认，以后还可以设置。

7.完成

这里了可以直接点击完成。可以之后来设置的。这样虚拟机就创建好了。

三、设置新建的虚拟机

1.打开设置页面

点击红框中任何一个都可以打开设置页面。

其他的设置都比较简单，这里主要说明一下以下两个设置：

2.设置安装镜像路径

选择好CentOsMini 的iso存放路径。就是把光驱设置成使用ISO挂载。

3.设置网络

选择好桥接模式。

四、安装CentOs

1.首先会让你选择进入方式，按默认进入就行了。之后会检查一些东东。

2.就进入了图形化配置的界面，首先是选择语言。

2.选择语言

3.设置安装位置.安装源也可设置，一般默认自动检测就行了。

4.设置安装位置，安装位置直接选新建虚拟机时建立的硬盘位置就行。

5.开启网络。

6.点击开始安装。

7.等待安装完毕页面。

8.设置root密码。

9.结束配置。（当然也可以创建用户，这里跳过）

10.重启

11.使用root账户登录
